## Как работать с flake8
1. Зачем нужна?
    * flake8 - нужен для проверки программного кода. Утилита проверяет правильно ли отфармотирован файл.
2. Как запустить?
    * В консоли прописываем `flake8 <name>` <name> - имя файла <b>с расширением</b>
